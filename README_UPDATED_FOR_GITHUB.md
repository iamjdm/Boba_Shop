## 💼 Featured Projects

### **[TeaZen Boba Bar](https://github.com/iamjdm/Boba_Shop_Senior_Seminar)** — Full-Stack Capstone Project

_Senior seminar project demonstrating modern full-stack web development_

A complete web platform for a boba shop featuring:

- **Backend:** Python Flask REST API with 10+ endpoints for menu, orders, events, and jobs
- **Database:** MySQL with relational schema (7 tables), ORM via SQLAlchemy, referential integrity
- **Frontend:** Vanilla JavaScript with responsive design (mobile-optimized), Fetch API for async communication
- **AI Integration:** Ollama-powered chatbot assistant that answers menu/allergen/event questions using grounded knowledge base
- **Features:** Live menu browsing, shopping cart with checkout, event calendar with RSVP, job application portal, AI order summaries
- **Deployment:** Localhost with environment configuration, CORS handling, error management

**Tech Stack:** Flask | MySQL | SQLAlchemy | Vanilla JS | HTML5/CSS3 | Ollama LLM  
**Key Skills Demonstrated:** Database design, API development, AI/ML integration, full-stack debugging, responsive UI/UX
